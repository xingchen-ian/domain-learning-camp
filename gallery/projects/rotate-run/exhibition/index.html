旋转世界

Q / E 旋转整个场景，重力始终向下，用"翻世界"的方式把主角送到终点。

🧩

唯一路径

每一关都手工设计成只有一条通关路线，逼你想清楚再动手。

📈

渐进难度

前 6 关教学循序渐进引入移动、跳跃、地刺、齿轮与旋转，之后旋转通关占比越来越高。

怎么玩

  1. 1

    移动

    ← → 或 A / D 让主角沿当前"地面"左右移动。

  2. 2

    跳跃

    ↑ / W / 空格 逆着重力跳起，跨过缝隙与障碍。

  3. 3

    旋转世界

    Q 逆时针、E 顺时针把整个场景转 90°，重力仍朝下，地形随之改变。

  4. 4

    到达终点

    碰到发光的绿色传送门即过关。碰到地刺或齿轮会失败重来。
