# Third-party libraries

This project uses one third-party library, vendored locally.  No CDN
or remote asset is loaded at runtime.

## three.js

- **Library:** three.js
- **Version:** 0.160.0
- **License:** MIT
- **Source:** https://github.com/mrdoob/three.js/tree/r160
- **File:** `exhibition/assets/vendor/three.module.min.js`
- **Downloaded from:** https://unpkg.com/three@0.160.0/build/three.module.min.js
- **Used for:** rendering the 3D Rubik's cube, scene, camera, lights,
  raycasting, and the render loop.

## Fonts

The project uses the system font stack
(`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", ...`).
No web font is loaded.

## Audio

No audio assets are used in this version.
