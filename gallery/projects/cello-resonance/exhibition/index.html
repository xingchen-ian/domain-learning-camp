弦境回响

Cello Resonance — 2D叙事解谜冒险游戏

核心概念： 一个手持古董大提琴的探险者穿行于神秘森林， 用声音与世界互动——音高创造光桥、弓压碎石或生藤、和声共鸣驱散守护者。 这不是演奏模拟器，而是以大提琴为叙事工具的解谜冒险。

创意起源

大提琴的音域横跨近三个八度，从深沉的 C2 到明亮的 A4。 在这个游戏中，我们将这些音高映射为与环境互动的力量：

  • 低音 (C2-G2) — 温暖厚重的光桥，缓慢而坚定地跨越深渊
  • 中音 (G3-C4) — 平衡的力量，既可碎岩也可生藤
  • 高音 (E4-A4) — 幽蓝冷冽的光桥，轻盈而遥远

游戏机制

第一章：音高光桥

不同的音高创造不同颜色的光桥。低音=暖色桥，高音=冷色桥。 用鼠标左右拖拽运弓，上下位置控制音高，在峡谷中架桥前行。

第二章：弓压机关

重弓压=音波震荡碎裂岩石墙壁，轻弓压=温和音调催生发光藤蔓。 拖拽速度和力度决定弓压——快速猛拖碎石，缓慢轻拉生藤。

第三章：和声共鸣

面对森林守护者，需要匹配特定的和声模式。 用特定音高运弓来共鸣，逐层削弱守护者，直到森林苏醒。

操控方式

操作方式效果
运弓鼠标左右拖拽发出声音、产生光波
音高鼠标上下位置上=高音(冷色)，下=低音(暖色)
弓压拖拽力度/速度猛拖=重压(碎石)，轻拖=轻压(生藤)
移动A/D 或 ←→左右移动探险者
共鸣Space (Boss阶段)持续发出和声共鸣

反馈系统

  • 音波涟漪 — 运弓时从大提琴发出的彩色光波 — 音高→颜色，弓压→大小
  • 光桥成形 — 正确音高范围内运弓时光桥逐渐出现 — 桥的亮度随运弓持续增强
  • 屏幕震动 — 重弓压碎岩时的视觉冲击 — 画面抖动+岩石裂痕动画
  • 藤蔓生长 — 轻弓压时发光藤蔓逐节生长 — 从种子点向上生长+叶片绽放
  • 守护者光环 — Boss阶段匹配音高时守护者变色 — 金色共鸣光覆盖紫色守护者

系统设计

音频引擎

Web Audio API 合成大提琴音色：三角波基频 + 锯齿波谐波（弓压越高谐波越丰富） + 正弦波二次谐波。 加上2.5秒卷积混响模拟森林空间感。

物理引擎

简单的2D平台物理：重力、平台碰撞、光桥和藤蔓作为可交互的动态平台。 相机跟随玩家，视差滚动背景树层。

视觉风格

手绘风格（painterly）：软性渐变填充、噪声纹理叠加、雾气粒子、 多层视差树、发光效果用径向渐变模拟。整体色调偏暖土色+深绿+幽紫。
