In the Opera House

A vintage theatre escape game by Yuyue Ji

Play the Game

Game Idea

A blonde opera singer is locked inside the theatre after the final curtain. You control her as she walks between the dressing room and the hidden stage, find clues, collect tools, and escape through the side door before the building locks for the night.

Domain & Learning

Domain: Room escaping / finding the right tool at the right time.

Novice misconception: Players often rush to the stage curtain and try to lift it manually.

Core shift: Learn to read visual clues (mirror colors, poem), collect tools (lens, key), and chain them into a sequence: open drawer → fix spotlight → open curtain → reveal key → escape.

System Design Summary

  • Environment data: Color code on the mirror (red, blue, gold); invisible lens and poem inside the locked drawer; the stage is hidden until the singer walks onto it.
  • Player-controlled data: Singer position in each room, inventory items (lens, poem, key), current code entry, and spotlight angle.
  • Calculated results: Code correctness, room transitions, whether the beam hits the chair, remaining time.
  • Feedback: Items glow when useful, drawers open, the spotlight turns on, the curtain opens automatically, and the key appears only when the chair is lit.
  • Success: Escape through the side door before the 10-minute timer reaches zero.
  • Failure: Time runs out and the theatre locks.

Controls

Use Arrow keys or WASD to move the opera singer. Walk to the right edge of the dressing room to enter the stage, and to the left edge of the stage to return. Click objects to investigate or use them. Select colors for the drawer code in the correct order. Once the spotlight is fixed and turned on, use the on-screen arrows or Q / E to rotate the beam until it illuminates the chair on stage.

Built with HTML, CSS, and plain JavaScript — no build step required.
