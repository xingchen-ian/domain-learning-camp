How the game was built

Approach

The game was developed as a single self-contained Canvas project in plain JavaScript — no frameworks, no build tools. Iteration happened in short playable loops: add a system, test it in the browser, refine.

Milestones

  • Tile grid, player movement, and collision.
  • Enemy patrol + vision cone + sight blocking.
  • Weapons (melee + ranged), contact-only death, blood effects.
  • Three levels with safe spawn rooms, doors, and lab-themed cover.
  • Masks / loadout selection and HUD feedback.

What I learned

Small, readable feedback (a detection bar, a danger vignette, clear animations) teaches the player more than complex rules. And environment design — where doors and cover are placed — drives the whole difficulty curve.
