Development Process

This timeline is based on agent-development-log.md and records the actual human-AI collaboration, including failures and unfinished work.

Interaction 01 — Project Setup

Goal: Read the brief, set up project skeleton, decide view angle and scope.

Student decision: 3D view (Three.js + Vite, Track C), include both jump and 5-second-stop features.

What happened: Created project folders, copied brief and system graph, initialized dev log, installed Vite + Three.js.

AI influence: AI suggested top-down 2D for faster delivery; student chose 3D for a more immersive experience. AI suggested skipping jump and 5-sec-stop for v1; student chose to include both.

Interaction 02 — Core Game Implementation

Goal: Build the smallest complete game loop: 3D scene, skier, controls, physics, collision, 3 levels, HUD, win/lose.

What happened: Implemented all game modules (constants, player, obstacles, HUD, main). Created 5 HTML website pages. Wrote shared CSS.

Key decisions: Separate level select (not sequential escalation). Rocks can't be jumped over; trees can. S-turn drag formula: speed − carveDrag × |steerAngle|.

Open question: Testing and build step pending. Network errors interrupted progress multiple times.
