Every stage combines Agent development work with student design judgment. Move forward only after both are complete.

Dependency StrategyDependency Strategy

开发工具可以复杂，展览运行必须简单。选择能实现设计目标的最低依赖等级，而不是一律禁止或一律使用npm。

Development tools may be complex; exhibition runtime must be simple. Choose the lowest dependency level that can realize the design instead of always banning or always using npm.

Track A · No Build

HTML、CSS、原生JavaScript和Canvas 2D。适合大多数简单2D原型。HTML, CSS, plain JavaScript, and Canvas 2D for most simple 2D prototypes.

Track B · Local Vendor

只使用少量固定版本的浏览器库，并把库文件保存在项目中，不使用CDN。Use a small version-pinned browser library stored locally in the project, not a CDN.

Track C · npm + Build

Three.js、多个ES Modules或复杂加载器可以使用npm与Vite；提交前必须生成静态展览版本。Three.js, multiple ES Modules, or complex loaders may use npm and Vite; produce a static exhibition build before submission.

统一运行标准：最终iMac只打开已经构建好的 exhibition/。不在展览电脑上执行 npm install。Three.js与ES Module项目通过静态HTTP服务器运行，不以双击HTML为验收标准。

Shared runtime standard: The iMac opens the prebuilt exhibition/. Never run npm install on exhibition machines. Serve Three.js and ES Module projects through static HTTP; double-clicking HTML is not the acceptance standard.

Session OutcomesSession Outcomes

Session结束时，项目不只是一个游戏文件，而是一套可以展示、继续开发和研究其过程的完整材料。

By the end of the session, your project is not only a game file. It is a complete set of materials that can be shown, continued, and studied.

项目网站Project Website

包含创意、领域知识、系统设计、开发过程和游戏入口。Idea, domain knowledge, system design, process, and game access.

可玩游戏Playable Game

一个完整核心循环和2–3个递进挑战。One complete core loop and 2–3 progressive challenges.

开发日志Development Log

按时间记录重要交互、失败、测试、决定和反思。Chronological interactions, failures, tests, decisions, and reflections.

源文件与素材Source and Assets

HTML、CSS、JavaScript、系统图、图片和声音。HTML, CSS, JavaScript, graph, images, and sounds.

README

运行方法、操作方式、文件结构、依赖和已知问题。Run instructions, controls, structure, dependencies, and known issues.

学生反思Student Reflection

哪些决定属于学生，AI如何影响方向，游戏是否仍表达原目标。Student decisions, AI influence, and fidelity to the original goal.

可离线运行的ZIP、manifest、预览图和全部项目文件。Offline-ready ZIP, manifest, preview image, and complete project files.
